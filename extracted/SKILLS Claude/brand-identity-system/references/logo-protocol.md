# Logo Protocol

Use this when developing or evaluating logo concepts.

## Choose the mark type

Select based on the brand problem:
- wordmark for names that can carry the identity alone
- combination mark for most expert-led businesses
- monogram only when the initials are already meaningful
- emblem only when containment or authority is strategically useful

## The big five tests

Every logo direction should pass:
1. first-read clarity
2. monochrome survival
3. small-size legibility
4. category distinctiveness
5. extension into web, slides, and documents

## Symbol discipline

- Use one metaphor stack, not five.
- Prefer symbols that compress the brand logic, not symbols that merely decorate it.
- Remove any element that only adds atmosphere.

## Typography logic

- Choose type based on authority structure, not trend.
- Explain whether the typeface signals institutional gravity, modern clarity, warmth, precision, or severity.
- Check spacing and weight before adding more graphic devices.

## Color logic

- Color should support recognition and emotional temperature.
- In trust-sensitive sectors, default to controlled contrast and disciplined accenting.
- If the mark dies in monochrome, the color system is compensating for weak form.

## Metaphor density

Ask:
- Is the symbol still understandable without explanation?
- Does each element carry strategic meaning?
- Is the metaphor stack too literary for its category?

## Simplification rule

When in doubt:
- remove one symbolic layer
- reduce one contrast effect
- simplify one contour

The burden is on complexity to prove its value.
