# LOTUS / Dr. Sort-Academic Helper plain-English rules
# Supported forms:
# move all files containing "phrase" to CATEGORY/doc_type
# tag all files containing "phrase" with tag1, tag2
# rename all files containing "phrase" as Template_{year}_{title}
# Available template fields: {year} {author} {title} {doc_type} {category} {language}

# Finance / admin examples
move all files containing "bank statement" to PERSONAL_ADMIN/tax_or_finance
move all files containing "notice of assessment" to PERSONAL_ADMIN/tax_or_finance
tag all files containing "invoice" with finance, billing, client_work
rename all files containing "invoice" as Invoice_{year}_{author}_{title}

# Scholarship examples
tag all files containing "bibliography" with research, references
rename all files containing "book review" as Review_{year}_{author}_{title}

# Creative / project examples
tag all files containing "screenplay" with creative, draft
