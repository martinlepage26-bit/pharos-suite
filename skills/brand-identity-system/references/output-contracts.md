# Output Contracts

Use the smallest contract that fits the request. Keep headings stable.

## Diagnostic review

- Verdict
- Strongest signal
- Strategic mismatch
- Operational risk
- Redesign priority
- Specific next moves

## Brand brief

- Audience
- Offer
- Differentiation
- Buying context
- Trust requirement
- Brand adjectives
- Avoid list
- Visual thesis

## Identity directions

Provide exactly 2 or 3 directions.

For each direction include:
- Direction name
- Premise
- Visual cues
- Why it fits
- Risks

## Logo concept sheet

- Concept name
- Logo type
- Form logic
- Typography logic
- Color logic
- Strengths
- Risks
- Usage behavior

## Website branding brief

- Hero mood
- Hierarchy
- Typography system
- Color system
- Layout rhythm
- Trust architecture
- Image logic
- Interaction tone

## Mini style guide

- Primary mark
- Alternates
- Palette
- Typography
- Spacing rules
- Dos and don'ts
